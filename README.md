<h1>Navbar Section</h1>

![Screenshot 2023-08-28 173321](https://github.com/mansi2020/css_major_project/assets/57188328/e710f143-7139-4cbd-8dde-46c9b6efee32)
<h1>HTML Description</h1>
nav: Represents a section of a web page that contains navigation links.

img: Represents an image element, used to display images on a web page.

i: Represents text that is set off from the normal text, often used for icons or for styling specific parts of text.

ul: Represents an unordered list, used to create a bulleted list of items.

li: Represents a list item within an ordered or unordered list.

a: Represents an anchor element, used to create hyperlinks that link to other web pages or resources.

<h1>CSS Description</h1>

padding: 1rem 5rem;: Sets padding for the top and bottom of the <nav> element as 1 rem and left and right as 5 rems.

display: flex;: Defines a flex container, allowing flexible layout of child elements.

position: fixed;: Positions the <nav> element as fixed, making it stay in a fixed position within the viewport as the user scrolls.

justify-content: space-between;: Aligns flex items along the main axis (horizontally) with space distributed between them.

align-items: center;: Aligns flex items along the cross axis (vertically) and centers them within the container.

background-color: var(--dark-green-color);: Sets the background color of the <nav> element to the value defined by the CSS variable "--dark-green-color".

width: 100%;: Sets the width of the <nav> element to 100% of its containing element.

top: 0;, left: 0;: Positions the <nav> element at the top left corner of its containing element.

z-index: 9;: Sets the stacking order of the <nav> element, making it appear above other elements with lower z-index values.

flex-wrap: wrap;: Specifies that the flex items can wrap onto multiple lines if they exceed the width of the container.

nav a: This targets all anchor (<a>) elements within the <nav> element.

text-decoration: none;: Removes the underline decoration from anchor links.

transition: color 0.3s ease-in-out;: Specifies a smooth color transition effect over 0.3 seconds when the color of the links changes.

clip-path: polygon(...): Applies a clip path to the element, defining a custom shape.

height: 1.4rem;, width: 1.6rem;: Sets the height and width of the element with the "social-icons" class.

nav img: This targets the images within the <nav> element.

height: 2rem;: Sets the height of the images to 2 rems.
